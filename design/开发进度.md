## 技术决策记录

### 决策 1: 配置系统使用 Pydantic

**时间**: 2026-03-19  
**决策**: 使用 Pydantic v2 作为配置验证和序列化库

**理由**:
- 类型安全
- 自动验证
- 良好的错误消息
- 支持嵌套配置
- 与 FastAPI 集成良好

**替代方案**:
- attrs + cattrs
- dataclasses + marshmallow

### 决策 2: 框架动态加载使用独立目录

**时间**: 2026-03-19  
**决策**: 使用 `--target` 参数安装框架到独立目录

**理由**:
- 避免污染全局环境
- 支持多版本共存
- 易于卸载和清理
- 支持热重载

**实现**:
- PyPI 框架安装到 `./frameworks/<name>`
- LLM SDK 安装到 `./llm_providers/<name>`

### 决策 3: 异步框架加载

**时间**: 2026-03-19  
**决策**: 使用 asyncio 实现异步框架加载

**理由**:
- 不阻塞主线程
- 支持并发加载多个框架
- 更好的用户体验
- 与现代 Python 实践一致
