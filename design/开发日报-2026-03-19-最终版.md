    max_memory_mb=256,
    network_enabled=False,

## 🎯 核心功能演示

### 端到端工作流程

```
1. 用户输入需求
   ↓
   "我想要一个销售数据可视化面板"

2. PM Agent 处理
   ↓
   - 意图识别：TaskType.FEATURE
   - 需求澄清（如需要）
   - 生成任务卡

3. 任务卡
   ↓
   TaskCard {
     type: feature,
     requirements: [
       {type: "ui", name: "SalesDashboard"},
       {type: "model", name: "SalesData"},
       {type: "api", name: "SalesAPI"}
     ]
   }

4. 代码生成器
   ↓
   - 生成 SQLAlchemy 模型
   - 生成 Pydantic Schema
   - 生成 FastAPI 路由
   - 生成 React 组件

5. 沙箱验证
   ↓
   - 代码安全扫描
   - Docker 容器执行
   - 验证功能正确性

6. 输出结果
   ↓
   - 保存生成的文件
   - 返回验证结果
   - 等待用户确认
```
