#### 3.2 PM Agent (100%)

**文件**: `backend/app/agents/pm_agent.py` (430 行)

**功能**:
- ✅ 自然语言需求理解
- ✅ 意图识别 (基于关键词)
- ✅ 多轮对话澄清
- ✅ 结构化任务卡生成
- ✅ 优先级自动判断
- ✅ 需求规格提取
- ✅ 会话历史管理

**工作流程**:
```
用户输入
    │
    ↓
意图识别 (Feature/Bugfix/Config/Refactor)
    │
    ↓
判断是否需要澄清 ──→ 是 ──→ 生成澄清问题
    │                      (最多 3 轮)
    │ 否
    ↓
生成结构化任务卡
    │
    ↓
输出 TaskCard
```

**使用示例**:
```python
from app.agents.pm_agent import create_pm_agent

# 创建 PM Agent
pm_agent = create_pm_agent()

# 处理用户消息
response = await pm_agent.process_message(
    session_id="session-123",
    message="我想要一个销售数据可视化面板",
)

# 响应包含澄清问题或任务卡
if response.metadata.get("needs_clarification"):
    print("需要澄清:")
    print(response.content)
else:
    task_card = response.metadata["task_card"]
    print(f"已生成任务：{task_card['id']}")
```

**意图识别示例**:
```python
# 用户输入："我想要一个销售数据可视化面板"
# 识别结果：TaskType.FEATURE

# 用户输入："修复登录页面的 bug"
# 识别结果：TaskType.BUGFIX

# 用户输入："配置数据库连接池大小"
# 识别结果：TaskType.CONFIG
```
