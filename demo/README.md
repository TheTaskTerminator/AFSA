# AFSA MVP 演示脚本

## 演示场景：用户说→界面变

### 场景描述
用户通过自然语言描述需求："给订单列表添加一个批量导出按钮"

---

## 演示步骤

### 步骤 1：打开演示页面 (30 秒)

1. 访问 `http://localhost:3000/demo`
2. 点击"进入演示模式"按钮
3. 界面切换到演示引导模式

**预期效果**：
- 页面顶部显示演示进度条（步骤 1/5）
- 右侧显示当前步骤说明卡片
- 对话区域显示预设的欢迎消息

---

### 步骤 2：输入自然语言需求 (30 秒)

1. 在对话输入框中输入：
   > "给订单列表添加一个批量导出按钮，支持导出为 Excel 格式"

2. 点击发送或按 Enter

**预期效果**：
- 消息显示在对话列表中
- 系统显示"正在分析需求..."状态
- PM Agent 开始工作，显示思考过程

---

### 步骤 3：AI 需求澄清对话 (1 分钟)

系统自动回复：

```

为了更准确地实现您的需求，我想确认几个细节：

1. 导出范围：导出当前页数据还是全部订单？
2. 导出字段：需要包含哪些字段？（订单号、客户名、金额、状态、创建时间）
3. 权限控制：是否需要限制特定角色才能使用导出功能？

您可以直接回答，或说"使用默认设置"继续。
```

用户回复："使用默认设置"

**预期效果**：
- 展示多轮对话澄清能力
- 显示 AI 理解需求的智能过程
- 任务卡片自动生成中

## 演示环境要求

- Node.js 18+
- Python 3.11+
- Docker Desktop
- 浏览器（推荐 Chrome）

## 启动命令

```bash
# 启动后端
cd backend
uv run uvicorn app.main:app --reload

# 启动前端
cd frontend
pnpm dev

# 访问演示页面
open http://localhost:3000/demo
```
